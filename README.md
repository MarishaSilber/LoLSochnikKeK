# VuzHub

Студенческая платформа взаимопомощи

## Запуск проекта

```bash
npm install
npm run dev
```

## Структура проекта

```
VuzHub/
├── src/
│   ├── app/
│   │   ├── components/     # UI компоненты
│   │   ├── pages/          # Страницы приложения
│   │   ├── data/           # Данные (студенты, фильтры)
│   │   ├── App.jsx         # Главный компонент
│   │   └── routes.jsx      # Роутинг
│   ├── styles/
│   │   └── index.css       # Основные стили
│   └── main.jsx            # Точка входа
├── index.html
├── package.json
└── vite.config.js
```

## Страницы

- **Главная** (`/`) - Поиск студентов, фильтры, карточки
- **Регистрация** (`/register`) - Форма регистрации
- **Профиль** (`/profile`) - Страница профиля студента
- **Поиск** (`/search`) - Расширенный поиск
- **Чат** (`/chat`) - Сообщения
- **Редактирование** (`/edit-profile`) - Редактирование профиля

## Технологии

- React 18
- React Router DOM
- Vite
