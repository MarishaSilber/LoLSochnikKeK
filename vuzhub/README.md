# ВузХаб — Платформа взаимопомощи студентов

Веб-приложение для помощи студентам в поиске поддержки на факультете.

## 🚀 Стек технологий

**Frontend:**
- React 19
- Vite 6
- React Router v7
- Tailwind CSS
- Radix UI
- Lucide React (иконки)
- Axios
- date-fns

**Backend (планируется):**
- Python (FastAPI/Flask)
- PostgreSQL + pgvector
- SQLAlchemy
- OpenRouter API (для умного парсинга запросов)

## 🛠️ Установка и запуск

### Требования
- Node.js 18+ 
- npm 9+

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

### Сборка для продакшена
```bash
npm run build
```

### Предпросмотр сборки
```bash
npm run preview
```

## 📱 Основные страницы

### 1. Главная (Поиск)
- Умная поисковая строка с AI-индикатором
- Фильтры (курс, локация, активность)
- Сетка карточек студентов с процентом совпадения
- Быстрые предложения запросов

### 2. Профиль
- Редактирование личной информации
- Загрузка аватара
- Описание «О себе» с авто-генерацией тегов
- Выбор локации (лаборатория, корпус)
- Статистика (рейтинг, количество помощей)

### 3. Мессенджер
- Список чатов
- Контекстные подсказки для быстрых ответов
- Индикатор онлайн-статуса
- Информация о собеседнике

## 🔌 API (планируется)

### Эндпоинты:
- `POST /api/search` — Умный поиск студентов
- `GET /api/students/:id` — Профиль студента
- `PUT /api/profile` — Обновление профиля
- `GET /api/chats` — Список чатов
- `GET /api/chats/:id/messages` — Сообщения чата
- `POST /api/messages` — Отправка сообщения

## 🧪 Моковые данные

Для тестирования интерфейса используются моковые данные в `src/lib/mockData.js`:
- 6 профилей студентов с разными компетенциями
- 3 чата с историей сообщений
- Профиль текущего пользователя

## 📊 Метрики ранжирования

Система оценивает совпадение по следующим параметрам:
1. **Course Match (x2.0)** — Совпадение курса
2. **Tag Overlap (x1.5)** — Количество совпавших тегов
3. **Text Rank (x1.0)** — Текстовое сходство bio
4. **Location Match (x0.8)** — Гео-близость
5. **Recency (x0.4)** — Актуальность (время последней активности)
6. **Trust Score (x0.3)** — Социальное доказательство (рейтинг)
